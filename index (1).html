Education

The Foundation

🏛️

El Shorouk Academy

B.Sc. Computer Science — Software Development Track
El Shorouk City, Cairo Faculty of Computer Science Software Development

My journey into software started long before college — with curiosity, a lot of late-night tutorials, and a growing obsession with understanding how things actually work under the hood. El Shorouk Academy gave that curiosity a structure. Studying Computer Science there didn't just teach me syntax or theory — it taught me how to think like an engineer.

Throughout my studies, I gravitated strongly toward backend development — the part of software that most people never see, but that makes everything else possible. I became fascinated by how a well-designed API can power an entire product, how proper database indexing can cut response time by an order of magnitude, and how clean architecture keeps a codebase maintainable as it grows in complexity.

The .NET ecosystem quickly became my home. C# felt natural — expressive, structured, and powerful. ASP.NET Core gave me everything needed to build real production-grade systems. SQL Server taught me that data design is as important as application code. Over time I expanded outward — picking up frontend skills in HTML, CSS, TypeScript, and JavaScript — becoming a developer who can own the full stack, not just a corner of it.

More than any specific technology, my education gave me a mindset: never stop asking why. Every project I've built since has been an extension of that — understanding deeply, building carefully, and always shipping something I'm proud of.

Training & Experience

Real-World Practice

🇪🇬
Digital Egypt Pioneers Initiative
DEPI — Ministry of Communications & IT, Egypt
In Progress — Trainee

I'm currently enrolled in the Digital Egypt Pioneers Initiative (DEPI), a national training program backed by Egypt's Ministry of Communications and Information Technology. My track is Full Stack .NET Developer — and it has been one of the most hands-on, practically focused learning experiences I've had.

Unlike purely academic study, DEPI puts you inside real development workflows from the very first week. I've been building and consuming APIs with ASP.NET Core, designing SQL Server databases with proper normalization and indexing strategies, and working with Entity Framework Core for clean, maintainable data access layers. The program is deeply rooted in industry practices — version control with Git, peer code reviews, structured debugging, and delivering working features under realistic constraints.

This experience has sharpened my ability to move from concept to working implementation faster, communicate technical decisions clearly, and think about systems as a whole — not just writing code, but understanding why the architecture is designed the way it is. It's the bridge between academic knowledge and professional readiness.
